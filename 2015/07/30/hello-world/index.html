博客安装

什么是Hexo?
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装前提

在git中使用npm install -g -hexo-cli 需要全局翻墙 最好自备一个 vpn 安装都是一键式
也可以去淘宝那下载镜像 NPM(Node) 镜像

  • Node.js
  • Git

如果您的电脑中已经安装上述必备程序，那么恭喜您！接下来只需要使用 npm 即可完成 Hexo 的安装。

进入网站需要hexo server开启 不关闭

完成安装后 127.0.0.1:4000（loaclhost:4000）进入本地的hexo网站
